# task-list

Desafio 
  * Adicionar novas tarefas;
  * Marcar e desmarcar o status de concluído;
  * Editar o conteúdo da task;
  * Deletar uma task;
  
Tecnologias e padrões utilizados
  - dotnet core 2.2
  - FluentValidation
  - RepositoryPattern
  - Commands - Handlers
  - EntityFramework (SQL Server)
  - JWT Bearer
  - Solid e Clean Code
  - Interface em Vue.js com Vuetify
